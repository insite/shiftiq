PivotTable Class

In the example above we have 3 Row Dimensions and 2 Column dimensions. Each dimension has at least one unit.

Row Dimensions

  1. ClientAgeType { Adult, Youth }
  2. ClientAge { 5-17 years, 18-64 years, 65+ years, 0-4 years, 0-17 years, 18+ }
  3. PracticeSettingType { Care, School/School Board }

Column Dimensions

  1. Taxonomy { Do, Know, Solve }
  2. Difficulty { Very Easy, Easy, Moderate, Difficult, Very Difficult }

Cells

Each cell is an integer value, indexed by row and column.

Row and column indexes looks like this: X|Y|Z

... where vertical bars are used to separate units.

For example, the cell value "4" in the top left of the pivot table would have this row :: column index:

Adult|5-17 years|Care :: Do|Easy
